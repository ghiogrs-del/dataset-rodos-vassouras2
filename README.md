

# Dataset – Rodos x Vassouras

Este repositório contém um dataset de imagens para classificação
entre rodos e vassouras, utilizado em atividades introdutórias
de Machine Learning e Visão Computacional.

## Estrutura
- rodos/: imagens de rodos
- vassouras/: imagens de vassouras

## Objetivo
Treinar modelos de classificação de imagens.
